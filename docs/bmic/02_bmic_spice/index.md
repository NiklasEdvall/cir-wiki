# Working on SPICE

If you want to work with PET data from BMIC on SPICE, CIR offers several tools to help you get started. We recommend you first read and follow the instructions on how to convert your data to BIDS following the documentation [here](https://k-cir.github.io/cir-wiki/bmic/02_bmic_spice/01_BIDS_on_SPICE)
